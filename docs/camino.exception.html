camino.exception

camino

Camino custom exceptions.

Note: Names are not postfixed with "Exception"; since we only catch and throw exceptions, that's redundant. I like the way it reads with the shorter names, especially when using the try/catch syntax; and it easily differentiates my exceptions from third-party exceptions.
