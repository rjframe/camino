Record.record

camino history Record

Provides access to the underlying JSONValue record.

This method is alias thised to the Record; operations should be typically performed directly on the Record object instead of explicitly via this method.

struct Record
pure nothrow @nogc @property ref
JSONValue
record
()

Examples

import std.json : JSONValue;
import std.stdio : writeln;

auto data = JSONValue(`{"2020-01-01": {}, "v": "1.0.0"}`);
auto record = Record(data, 0);

// These two statements are identical:
writeln(record.toPrettyString());
writeln(record.record().toPrettyString());
